# Student Management System Database

## Overview

A relational database project developed using MySQL to manage student, course, faculty, enrollment, and attendance information. The project demonstrates database design, normalization, data integrity, reporting, and analytics using SQL.

## Features

* Relational database with 5 interconnected tables
* Primary Key and Foreign Key constraints
* Normalization up to Third Normal Form (3NF)
* Attendance tracking system
* Enrollment management
* Reporting and analytics using SQL
* Reusable database views

## Database Schema

Tables included:

1. Students
2. Courses
3. Faculty
4. Enrollments
5. Attendance

## Technologies Used

* MySQL

* SQL
